## Output
- Nắm vững State trong React: useState, state bất biến, state lifting, khi nào component re-render
- Hiểu sâu về Dependency Array: dependency array rỗng [], có giá trị, không có array - khi nào effect chạy?
- useEffect: cleanup function, patterns (fetching data, subscriptions, event listeners)
- useMemo: khi nào dùng, memoization, dependency array trong useMemo
- useCallback: memoize functions, khi nào cần, kết hợp với React.memo
- React Tree Root: ReactDOM.createRoot(), component tree, root vs component tree
- useRef: mutable reference, accessing DOM, storing previous values
- useContext: Context API, Provider, khi nào dùng Context
- Component Patterns: composition, controlled vs uncontrolled components
